# Sri_Vaarahi_designs

## Backend build commands

Run the following commands from the Backend directory:

- Compile the project:
  - `./mvnw compile`
- Run the test suite:
  - `./mvnw test`
- Build the application and install it locally:
  - `./mvnw clean install`

If you are using a specific Java runtime, set it before running Maven:

- Java 21:
  - `export JAVA_HOME="/Users/sreekarchowdhary/.jdk/jdk-21.0.10/jdk-21.0.10+7/Contents/Home"`
  - `export PATH="$JAVA_HOME/bin:$PATH"`
